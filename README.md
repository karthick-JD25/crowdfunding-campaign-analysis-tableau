## Project Overview
This project analyzes crowdfunding campaign data to identify key factors influencing project success. The dashboard provides insights into funding trends, backer behavior, project categories, and success rates.

The objective is to help stakeholders understand patterns that contribute to successful fundraising campaigns.

## Problem Statement
Many crowdfunding campaigns fail due to poor planning, unrealistic funding goals, and lack of understanding of customer contribution behavior.

Data-driven insights can help improve campaign strategy and increase funding success rates.

## Tools Used
Tableau  
Excel  
sql

## Key Insights
• Product Design and Technology categories show higher success rates  
• Campaigns with moderate funding goals have better success probability  
• Backer contribution significantly impacts project success  
• Certain locations show higher campaign performance  
• Project duration influences funding outcomes

## Dashboard Features
• KPI metrics (Total backers, funding amount, success rate)
• Category-based project performance
• Location-wise funding analysis
• Year-wise campaign trend analysis
• Backer contribution insights

## Skills Demonstrated
• Tableau Dashboard Development  
• Data Visualization  
• Business Insight Generation  
• Trend Analysis  
• KPI Metrics Interpretation  
• Data Storytelling  

## Business Impact
The insights help entrepreneurs understand factors influencing successful campaigns. This supports better planning of funding goals, campaign duration, and category selection to improve success probability.
